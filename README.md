# cybergear control project with STM32H7
## Hardware
- STM32H743VIT6
- MCP2562FDT-E/SN
- NX5032GA-8.000M-STD-CSU-1

## Motor setting
Please set CAN ID of cybergear as 0x7f or 0x7e.
